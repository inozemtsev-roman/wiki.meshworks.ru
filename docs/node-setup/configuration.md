## Первоначальная настройка ноды

При первом запуске ноды первым делом потребуется выбрать **регион (Region)**. Эта страница — практическая конфигурация Meshtastic для России и типовых сетей. Этот параметр определяет, на каких частотах и с какими ограничениями будет работать радиомодуль. Региональные настройки напрямую влияют на дальность связи, стабильность сети и законность использования оборудования.
:::info
Выбираем регион **Russia**.
:::

Далее нам необходимо дополнительно настроить ноду. Настройки могут зависеть от региона и города использования.

## Москва / Московская область

С **16.04.2026** сеть в Москве и области использует пресет **Medium Fast**.

| Параметр | Значение | Комментарий |
|----|----|----|
| **Region** | RU | Региональная настройка для России |
| **Channels Preset** | MediumFast | Используемый местной сетью пресет |
| **POSIX Timezone** | MSK-3 | Московское время |
| **LORA / Modem preset** | MEDIUM_FAST | Используемый местной сетью пресет |
| **LORA / Hop limit** | 7 | Максимальное количество ретрансляций |
| **LORA / TX power** | 20 dBm (SX127x) / 22 dBm (SX126x) | При включённой опции **LAR** SX127x и SX126x это модели чипа LoRa |
| **LORA / Freq slot** | 2 |    |
| **LORA / Freq** | 869.075 МГц | Частота выставляется автоматически при выборе региона RU |

## Стандартные настройки

Почти во всех городах используются стандартные настройки и пресет **Long Fast**  
Если ваш город или регион использует другой пресет - сообщите нам, актуализируем данные на этой странице  

&nbsp;



| Параметр | Значение | Комментарий |
|----|----|----|
| **Region** | RU | Региональная настройка для России |
| **Channels Preset** | LongFast | Часто используется по умолчанию, но не всегда |
| **POSIX Timezone** | Зависит от региона | Укажите местный часовой пояс |
| **LORA / Modem preset** | LONG_FAST | Часто используется по умолчанию, но не всегда |
| **LORA / Hop limit** | 7 | Максимальное количество ретрансляций |
| **LORA / TX power** | 20 dBm (SX127x) / 22 dBm (SX126x) | При включённой опции **LAR** SX127x и SX126x это модели чипа LoRa |
| **LORA / Freq slot** | 2 |    |
| **LORA / Freq** | 869.075 МГц | Частота выставляется автоматически при выборе региона RU |
